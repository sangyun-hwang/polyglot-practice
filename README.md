# Polyglot Practice

> 같은 요구사항을 다양한 백엔드 스택으로 구현합니다.

## 📌 프로젝트 소개

이 레포지토리는 하나의 애플리케이션을 여러 언어와 프레임워크로 구현하며  
백엔드 구조와 개발 방식의 차이를 비교하기 위한 프로젝트입니다.

## 🧩 구성

- Next.js + Supabase (기본 구현)
- Express
- NestJS
- Spring Boot

## 🎯 목표

- 백엔드 아키텍처 이해
- 인증 / 라우팅 / 데이터 처리 방식 비교
- 언어별 개발 경험 차이 분석
